# wc-ui

🎨 面向未来的原生 Web Components UI组件库

  • 跨框架。无论是react、vue还是原生项目均可使用。
  • 组件化。shadow dom真正意义上实现了样式和功能的组件化。
  • 类原生。一个组件就像使用一个div标签一样。
  • 无依赖。纯原生，无需任何预处理器编译。
